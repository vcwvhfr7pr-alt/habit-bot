# Бот напоминаний для трекера привычек

## Как запустить — пошагово

### Шаг 1. Загрузите бота на GitHub

1. На github.com создайте **новый репозиторий** — назовите `habit-bot`
2. Загрузите три файла: `bot.py`, `requirements.txt`, `Procfile`

### Шаг 2. Запустите на Railway (бесплатно)

1. Зайдите на [railway.app](https://railway.app)
2. Нажмите **Start a New Project**
3. Выберите **Deploy from GitHub repo**
4. Подключите GitHub и выберите репозиторий `habit-bot`
5. Railway сам найдёт `Procfile` и запустит бота

### Шаг 3. Добавьте токен бота

После деплоя нужно указать токен:

1. В Railway откройте ваш проект
2. Перейдите во вкладку **Variables**
3. Нажмите **New Variable** и добавьте:
   - Имя: `BOT_TOKEN`
   - Значение: токен вашего бота (от @BotFather)
4. Railway автоматически перезапустит бота

### Шаг 4. Проверьте

Напишите вашему боту в Telegram:
- `/start` — должно появиться приветствие
- `/remind 09:00` — установить напоминание на 9 утра

## Команды бота

| Команда | Что делает |
|---|---|
| `/remind 09:00` | Установить напоминание на 9:00 каждый день |
| `/remind_off` | Отключить напоминание |
| `/status` | Посмотреть текущее время напоминания |

## Важно про часовой пояс

По умолчанию время на сервере Railway — UTC (Лондон, +0).
Если вы в Москве (UTC+3) и хотите напоминание в 9:00 по Москве —
установите `/remind 06:00` (вычтите свой часовой пояс).

Или добавьте переменную окружения `TZ=Europe/Moscow` в Railway Variables —
тогда можно вводить московское время напрямую.
